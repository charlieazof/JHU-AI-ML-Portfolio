# JHU AI/ML Portfolio

This repository contains the projects I developed as part of the 
**Johns Hopkins University Diploma in Artificial Intelligence** program. 
Each project tackles a real-world problem using a different family of 
AI/ML techniques.

## Projects

### ✈️ 01 - Airline Delay Analysis
Exploratory data analysis and predictive modeling to identify the key 
drivers of flight delays and forecast delay likelihood based on 
operational, temporal, and weather-related features.  
*Techniques:* EDA, feature engineering, classification/regression models.

### 💳 02 - Credit Card Fraud Detection
Detection of fraudulent transactions on a highly imbalanced dataset, 
applying resampling strategies and supervised learning to maximize 
recall while controlling false positives.  
*Techniques:* imbalanced classification, SMOTE/undersampling, 
precision-recall evaluation.

### 🧠 03 - Brain Tumor Detection
Image classification of MRI scans to detect the presence of brain 
tumors, using convolutional neural networks and transfer learning 
on medical imaging data.  
*Techniques:* deep learning, CNNs, transfer learning, computer vision.

### ⚡ 04 - RAG for the Energy Industry
A Retrieval-Augmented Generation pipeline that answers domain-specific 
questions from energy-sector documents, combining a vector database 
with a large language model.  
*Techniques:* embeddings, vector search, LLM prompting, RAG architecture.

## Tech Stack
Python · scikit-learn · TensorFlow/PyTorch · pandas · LangChain · 
vector databases (FAISS/Chroma) · Jupyter

## About
These projects reflect my journey through the JHU AI program and 
my interest in applying AI to practical business and scientific problems.
